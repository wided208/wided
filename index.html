Quelle est le but de COSESI?

« COSESI » est une application desktop développée dans le cadre du projet pluridisciplinaire de 2CP.

Ce logiciel a pour but d’aider la commission des œuvres sociales d’esi de suivre les différents types des prêts d’une manière plus facile et rapide.

Comment utiliser l'application ?

1-Se connecter à l'internet de l'esi.
2-se connecter via l'application.
3- choisir : s’inscrire, Connecter ou Quitter.

Comment s'inscrire ?

1-Cliquer sur «s’inscrire »
2-Enter un nom d’utilisateur, un mot de passe et l'adresse email.
3-accepter les conditions d’utilisation.

Comment connecter via l'application ?

1-Cliquer sur «Connecter».
2-Entrer votre identifiant et votre mot de passe .
3-Cliquer sur «Log in» pour valider.

Comment ajouter une demande ?

1-Cliquer à gauche sur le menu et choisir Demandes et PVs puis cliquer sur ajouter une demande
2-Entrer le Nom, prénom et le numéro de téléphone de la personne qui a demandé le prêt social, ainsi que la date de la demande.
3-Enter le type de pret et son motif ainsi que le montant du pret demandé.
4-Dans le cas d'un prét anticipé, entrer aussi le salaire par mois, la prime sociale et le pret anticipé par espace.
5-Cliquer sur ajouter une demande.

Comment consulter l'Historique d' un pret ?

1- Afficher nombre de préts par employé : En cliquant sur l'Historique dans le menu puis sur afficher le nombre des préts par employé, séléctionner l'employé qui convient puis cliquer sur afficher pour afficher le resultat.

2-Afficher nombre de préts par service : En cliquant sur l'Historique dans le menu puis sur afficher le nombre des préts par employé, séléctionner le service qui convient puis cliquer sur afficher pour afficher le resultat.

3- Afficher nombre de préts par type : En cliquant sur l'Historique dans le menu puis sur afficher le nombre des préts par employé, séléctionner le type qui convient puis cliquer sur afficher pour afficher le resultat.

4-Afficher nombre de préts par période : En cliquant sur l'Historique dans le menu puis sur afficher le nombre des préts par employé, séléctionner la période qui convient puis cliquer sur afficher pour afficher le resultat.
